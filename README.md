# 🎮 Guess the Number Game

This is a simple Python project where the computer picks a random number and the player must try to guess it. It was one of my first projects to practice programming logic and loops!

## 🚀 How to Play

1. Make sure you have **Python** installed on your machine.
2. Download the `guess_number.py` file.
3. Open your terminal in the folder where the file is located and type:
   ```bash
   python guess_number.py
🛠️ Features
Random Number Generation: Uses the random library to choose a target.

Real-time Feedback: Tells you if your guess was too high or too low.

Attempt Counter: Tracks how many tries it took you to win.

🧠 What I Learned:
Module Imports: Working with import random.

Loops: Using while loops to keep the game running.

Conditionals: Logical flow with if, elif, and else.

Input Handling: Capturing and converting user data.
